# AI Skills Configuration & System Prompts

Усі системні промпти для нашого AI-асистента версіонуються у цьому репозиторії.

## Поточні скіли
* `cv-extraction` — парсинг та виділення жорстких навичок (Hard Skills) з резюме.
* `job-match-scoring` — математичний розрахунок відповідності кандидата вимогам вакансії.

## Правило релізу (SDLC)
Будь-яка зміна тексту інструкцій для моделей повинна проходити через Pull Request із обов'язковим успішним проходженням тестів із папки `evals/`.
