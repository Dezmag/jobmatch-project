# Критерії оцінювання якості Cover Letter (LLM-as-a-Judge)

Кожен згенерований супровідний лист (Cover Letter) оцінюється за шкалою від 0 до 3 за трьома осями. Baseline для проходження CI: загальний середній бал не нижче **2.5 / 3.0**.

### 1. Релевантність (Relevance)
- **3 бали**: Лист чітко підкреслює навички кандидата, які вимагає вакансія.
- **0 балів**: Лист загальний або не згадує технології з вакансії.

### 2. Тон (Tone)
- **3 бали**: Професійний, переконливий, діловий та впевнений стиль.
- **0 балів**: Занадто фамільярний, агресивний або неформальний тон.

### 3. Відсутність галюцинацій (Faithfulness / No Hallucinations)
- **3 бали**: ШІ використовує виключно факти з наданого резюме (CV).
- **0 балів**: ШІ вигадав досвід роботи, компанії або навички, яких немає в резюме.
