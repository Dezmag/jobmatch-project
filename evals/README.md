# AI Prompt Evaluation Suite (LLMOps)

Ця папка містить систему автоматичного тестування якості системних промптів додатка JobMatch.

## Як це працює
1. `dataset.json` зберігає еталонні кейси (резюме + вакансії).
2. Скрипт `run-evals.js` автоматично проганяє тексти через логіку промптів у CI-пайплайні.
3. Метрика **Accuracy** має становити 100% для успішного злиття Pull Request.
